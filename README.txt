■WebHive構成
---------------------------------------------------------------------------

WebHiveを導入するサーバにapache,PHP,hive clientを導入し適切な設定を行ってください。
WebHiveはMySQL(PostgreSQLでも可能)上にWebHiveリポジトリを保持しています。


　　WEB Browser
　　　　｜
　　　(http)
　　　　｜
　　　Apache
　　　　｜
　　　WebHive　－－－－－－　MySQL(WebHive Repository)
　　　　｜
　　hive client　－－－－－　MySQL(Hive Repository)
　　　　｜
　Hadoop(HDFS/MapRFS)
